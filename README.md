## 🚀 Memulai

### Apa itu UndanginAja?

UndanginAja adalah platform digital yang memungkinkan Anda:
- ✅ Membuat undangan pernikahan digital yang indah

## 🎨 Membuat Undangan

### Langkah-langkah Membuat Undangan

#### 1. Pilih Template

1. **Masuk ke Dashboard** setelah login
2. **Lihat koleksi template** yang tersedia
3. **Klik "Preview"** untuk melihat template
4. **Klik "Pilih Template"** pada template yang diinginkan

#### 2. Isi Detail Undangan

Setelah memilih template, isi informasi berikut:

**Informasi Mempelai:**
- Nama mempelai pria
- Nama mempelai wanita
- Nama orang tua (opsional)

**Detail Acara:**
- Tanggal acara
- Waktu acara
- Lokasi/venue
- Alamat lengkap

**Informasi Tambahan:**
- Pesan khusus
- Dress code (opsional)
- Kontak person

#### 3. Preview dan Simpan

1. **Klik "Preview"** untuk melihat hasil undangan
2. **Periksa semua informasi** sudah benar
3. **Klik "Simpan"** untuk menyimpan undangan
4. **Undangan berhasil dibuat!**

### Tips Membuat Undangan yang Menarik

- ✅ Gunakan foto berkualitas tinggi
- ✅ Periksa ejaan nama dan tanggal
- ✅ Pilih warna yang sesuai tema pernikahan
- ✅ Tambahkan pesan personal yang menyentuh
- ✅ Pastikan informasi lokasi lengkap dan jelas

## 📧 Mengirim Undangan

### Kirim via Email

#### 1. Akses Fitur Communication

1. **Klik "Communication"** pada undangan
2. **Pilih tab "Email"**

#### 2. Setup Email

1. **Masukkan daftar email** (pisahkan dengan koma)
2. **Tulis subject email** (opsional, akan ada default)
3. **Tulis pesan personal** (opsional)
4. **Centang "Attach PDF"** untuk melampirkan undangan

#### 3. Kirim Email

1. **Klik "Send Email"**
2. **Tunggu proses pengiriman**
3. **Konfirmasi email berhasil dikirim**

**Template Email Otomatis:**
```
Subject: Undangan Pernikahan [Nama Mempelai]

Dengan hormat,
Kami mengundang Anda untuk hadir dalam acara pernikahan kami.

Detail acara terlampir dalam file PDF.

Terima kasih,
[Nama Mempelai]
```

### Kirim via WhatsApp

#### 1. Generate Link WhatsApp

1. **Pilih tab "WhatsApp"** di Communication
2. **Masukkan nomor WhatsApp** (format: 08123456789)
3. **Tulis pesan custom** (opsional)
4. **Klik "Generate Link"**

#### 2. Bagikan Link

1. **Copy link WhatsApp** yang dihasilkan
2. **Klik link untuk membuka WhatsApp**
3. **Pesan otomatis akan muncul**
4. **Kirim pesan ke kontak**

**Template Pesan WhatsApp:**
```
Assalamualaikum/Selamat pagi,

Dengan hormat, kami mengundang Anda untuk hadir dalam acara pernikahan kami:

👰🤵 [Nama Mempelai]
📅 [Tanggal Acara]
🕐 [Waktu Acara]  
📍 [Lokasi Acara]

Link undangan: [URL Undangan]

Terima kasih atas perhatiannya.
```

### Bulk Sending

**Untuk Email:**
- Masukkan hingga 50 email sekaligus
- Pisahkan dengan koma atau enter
- Sistem akan validasi format email

**Untuk WhatsApp:**
- Generate link untuk setiap nomor
- Copy dan paste ke WhatsApp Web/Desktop
- Gunakan fitur broadcast WhatsApp

## ⚙️ Instalasi

### Persyaratan Sistem

**Server Requirements:**
- PHP 8.1 atau lebih baru
- MySQL 8.0 atau MariaDB 10.3
- Composer 2.0+
- Node.js 16+ dan npm
- Web server (Apache/Nginx)

**PHP Extensions:**
- BCMath PHP Extension
- Ctype PHP Extension
- Fileinfo PHP Extension
- JSON PHP Extension
- Mbstring PHP Extension
- OpenSSL PHP Extension
- PDO PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension
- GD PHP Extension (untuk image processing)

### Langkah Instalasi

#### 1. Clone Repository

```bash
git clone https://github.com/username/UndanginAja.git
cd UndanginAja
```

#### 2. Install Dependencies

```bash
# Install PHP dependencies
composer install

# Install Node.js dependencies
npm install
```

#### 3. Environment Setup

```bash
# Copy environment file
cp .env.example .env

# Generate application key
php artisan key:generate
```

#### 4. Database Configuration

Edit file `.env`:
```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=undanginaja
DB_USERNAME=root
DB_PASSWORD=your_password
```

#### 5. Mail Configuration

```env
MAIL_MAILER=smtp
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USERNAME=your_email@gmail.com
MAIL_PASSWORD=your_app_password
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS=your_email@gmail.com
MAIL_FROM_NAME="UndanginAja"
```

#### 6. Run Migrations

```bash
# Create database tables
php artisan migrate

# Seed sample data (optional)
php artisan db:seed
```

#### 7. Build Assets

```bash
# Build CSS and JS
npm run build

# For development
npm run dev
```

#### 8. Storage Setup

```bash
# Create storage link
php artisan storage:link

# Set permissions
chmod -R 755 storage
chmod -R 755 bootstrap/cache
```

#### 9. Start Server

```bash
# Development server
php artisan serve

# Access: http://localhost:8000
```

### Production Deployment

#### Environment Production

```env
APP_ENV=production
APP_DEBUG=false
APP_URL=https://yourdomain.com

# Database production
DB_HOST=your_production_host
DB_DATABASE=your_production_db
DB_USERNAME=your_production_user
DB_PASSWORD=your_secure_password

# Mail production
MAIL_HOST=your_smtp_host
MAIL_USERNAME=your_production_email
MAIL_PASSWORD=your_secure_password
```

#### Optimasi Production

```bash
# Cache configuration
php artisan config:cache

# Cache routes
php artisan route:cache

# Cache views
php artisan view:cache

# Optimize autoloader
composer install --optimize-autoloader --no-dev
```

## 🔒 Keamanan

### Fitur Keamanan

**Authentication & Authorization:**
- Laravel Sanctum untuk API authentication
- Role-based access control (Admin/User)
- Password hashing dengan bcrypt
- Session management yang aman

**Input Validation:**
- CSRF protection pada semua form
- XSS protection dengan HTML purification
- SQL injection prevention dengan Eloquent ORM
- File upload validation (type, size, content)

**Rate Limiting:**
- Login attempts (5 attempts per 5 minutes)
- API requests (60 per minute)
- File uploads (50 per hour per user)
- Broadcast creation (5 per hour per admin)

**Data Protection:**
- Encrypted database connections
- Secure file storage dengan proper permissions
- Personal data anonymization options
- GDPR compliance features

### Best Practices

**Untuk Admin:**
- Gunakan password yang kuat (minimal 12 karakter)
- Enable two-factor authentication jika tersedia
- Regular backup database dan files
- Monitor user activities dan suspicious behavior
- Update sistem secara berkala

**Untuk User:**
- Jangan share kredensial login
- Logout setelah selesai menggunakan
- Gunakan password yang unik
- Laporkan aktivitas mencurigakan ke admin

```bash
chmod -R 755 storage bootstrap/cache
chown -R www-data:www-data storage bootstrap/cache

# Clear cache
php artisan cache:clear

php artisan view:clear

# Check logs
tail -f storage/logs/laravel.log
```

#### 2. PDF Generation Error

**Penyebab:**
- Memory limit terlalu kecil
- Font tidak tersedia
- CSS tidak kompatibel

**Solusi:**
```bash
# Test PDF generation
php artisan pdf:test

# Clean temporary files
php artisan pdf:cleanup-temp

# Check memory limit in php.ini
memory_limit = 512M
```

#### 3. Email Tidak Terkirim

**Penyebab:**
- SMTP configuration salah
- Firewall blocking port 587
- Gmail app password tidak valid

**Solusi:**
```bash
# Test email configuration
php artisan tinker
>>> Mail::raw('Test email', function($msg) { $msg->to('test@example.com')->subject('Test'); });

# Check mail logs
tail -f storage/logs/laravel.log | grep mail
```

#### 4. Upload File Gagal

**Penyebab:**
- File size melebihi limit
- Storage permission error
- Disk space penuh

**Solusi:**
```bash
# Check disk space
df -h

# Check upload limits in php.ini
upload_max_filesize = 10M
post_max_size = 10M

# Fix storage permissions
chmod -R 755 storage/app/public
```

#### 5. Database Connection Error

**Penyebab:**
- Database credentials salah
- MySQL service tidak running
- Database tidak exists

**Solusi:**
```bash
# Check MySQL service
sudo systemctl status mysql

# Test database connection
php artisan tinker
>>> DB::connection()->getPdo();

# Create database if not exists
mysql -u root -p
mysql> CREATE DATABASE undanginaja;
```
