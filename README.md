### Persyaratan Sistem

**Server Requirements:**
- PHP 8.1 atau lebih baru
- MySQL 8.0 atau MariaDB 10.3
- Composer 2.0+
- Node.js 16+ dan npm
- Web server (Apache/Nginx)

**PHP Extensions:**
- BCMath PHP Extension
- Ctype PHP Extension
- Fileinfo PHP Extension
- JSON PHP Extension
- Mbstring PHP Extension
- OpenSSL PHP Extension
- PDO PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension
- GD PHP Extension (untuk image processing)

### Langkah Instalasi

#### 1. Clone Repository

```bash
git clone https://github.com/username/UndanginAja.git
cd UndanginAja
```

#### 2. Install Dependencies

```bash
# Install PHP dependencies
composer install

# Install Node.js dependencies
npm install
```

#### 3. Environment Setup

```bash
# Copy environment file
cp .env.example .env

# Generate application key
php artisan key:generate
```

#### 4. Database Configuration

Edit file `.env`:
```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=undanginaja
DB_USERNAME=root
DB_PASSWORD=your_password
```

#### 5. Mail Configuration

```env
MAIL_MAILER=smtp
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USERNAME=your_email@gmail.com
MAIL_PASSWORD=your_app_password
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS=your_email@gmail.com
MAIL_FROM_NAME="UndanginAja"
```

#### 6. Run Migrations

```bash
# Create database tables
php artisan migrate

# Seed sample data (optional)
php artisan db:seed
```

#### 7. Build Assets

```bash
# Build CSS and JS
npm run build

# For development
npm run dev
```

#### 8. Storage Setup

```bash
# Create storage link
php artisan storage:link

# Set permissions
chmod -R 755 storage
chmod -R 755 bootstrap/cache
```

#### 9. Start Server

```bash
# Development server
php artisan serve

# Access: http://localhost:8000
```

### Production Deployment

## ğŸ”§ Troubleshooting

### Masalah Umum

#### 1. Error 500 - Internal Server Error

**Penyebab:**
- Permission folder salah
- Environment configuration error
- Database connection error

**Solusi:**
```bash
# Fix permissions
chmod -R 755 storage bootstrap/cache
chown -R www-data:www-data storage bootstrap/cache

# Clear cache
php artisan cache:clear
php artisan config:clear
php artisan view:clear

# Check logs
tail -f storage/logs/laravel.log
```

#### 2. PDF Generation Error

**Penyebab:**
- Memory limit terlalu kecil
- Font tidak tersedia
- CSS tidak kompatibel

**Solusi:**
```bash
# Test PDF generation
php artisan pdf:test

# Clean temporary files
php artisan pdf:cleanup-temp

# Check memory limit in php.ini
memory_limit = 512M
```

#### 3. Email Tidak Terkirim

**Penyebab:**
- SMTP configuration salah
- Firewall blocking port 587
- Gmail app password tidak valid

**Solusi:**
```bash
# Test email configuration
php artisan tinker
>>> Mail::raw('Test email', function($msg) { $msg->to('test@example.com')->subject('Test'); });

# Check mail logs
tail -f storage/logs/laravel.log | grep mail
```

#### 4. Upload File Gagal

**Penyebab:**
- File size melebihi limit
- Storage permission error
- Disk space penuh

**Solusi:**
```bash
# Check disk space
df -h

# Check upload limits in php.ini
upload_max_filesize = 10M
post_max_size = 10M

# Fix storage permissions
chmod -R 755 storage/app/public
```

#### 5. Database Connection Error

**Penyebab:**
- Database credentials salah
- MySQL service tidak running
- Database tidak exists

**Solusi:**
```bash
# Check MySQL service
sudo systemctl status mysql

# Test database connection
php artisan tinker
>>> DB::connection()->getPdo();

# Create database if not exists
mysql -u root -p
mysql> CREATE DATABASE undanginaja;
```
